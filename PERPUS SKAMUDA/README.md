# Perpustakaan SMK — Sistem Peminjaman (Demo)

Instruksi singkat:

1. Buka folder proyek di VS Code.
2. Install ekstensi **Live Server** jika belum ada.
3. Klik kanan `index.html` → Open with Live Server, atau jalankan Live Server dari status bar.
4. Aplikasi menggunakan `localStorage` untuk menyimpan data (tidak ada backend).

Fitur:

- Menampilkan daftar buku
- Menambah peminjaman (pilih buku, nama, tanggal)
- Mengembalikan buku
- Pencarian buku

Catatan: ini adalah demo front-end sederhana untuk SMK; sesuaikan dan tambahkan backend sesuai kebutuhan.
